# SUFEDocs Archive

**SUFEDocs** 系上海财经大学（下称“上财”）校内资料分享平台，旨在使校内学生更方便地获取与课程有关的学习资源。

本项目二次开发自北京邮电大学 BYR Docs 以 [MIT 协议](https://opensource.org/license/MIT) 贡献的
[开源项目](https://github.com/byrdocs/byrdocs)
，在此我们表示最衷心的感谢。

## 注意事项

> [!WARNING]
> **本项目仅面向上财在校学生提供服务。** 如果您不是上财在校学生，我们保留中止向您提供服务的权利。

## 如何使用

您可以访问 [SUFEDocs](https://docs.xsufe.com) 网站，搜索并获取您想要的资源。

您可使用关键词，诸如书名、作者、出版社、ISBN、科目、年份来查找您想要的文件，也可直接选择某一分类之下的文件。

如果您找到了目标文件，点击下载按钮即可获取；如果是维基真题的题目，点击标题即可跳转。

## 如何贡献

- 贡献文件，请参阅[贡献指南](./CONTRIBUTING.md)
- 我们不接受任何形式的捐赠，除非是向我们提供资料。当然，您需要确保您对自己提供的资料有合法的分发权限。
- 如果您有意参与维护本项目，可发送邮件到 contact#xsufe.com (以@代替井号) 与我们联系。
